# Duck Cloud News

Duck Cloud News is a responsive and SEO-friendly news website. It dynamically loads news articles from a JSON file and allows users to read full articles.

## Features
- Responsive design
- SEO-friendly structure
- Dynamic content loading
- "Read More" functionality for full articles

## Technologies Used
- HTML
- CSS
- JavaScript
- JSON
- Markdown

## How to Use
1. Clone the repository.
2. Host the project on a static hosting platform like Vercel.
3. Update `data/data.json` to add or modify news articles.

## Deployment
This project is designed to be hosted on GitHub Pages or Vercel for easy deployment.
